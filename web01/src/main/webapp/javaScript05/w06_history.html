** history 객체 활용 **

** 이해를 돕는 단어 **
=> 속성 
  * attribute : Tag 의 속성
   -> 1. 속성(屬性), 특질
   -> 2. 부수물, 딸린 물건, 붙어 다니는 것
   -> 3. 속성·성질을 나타내는 말
   => JS를 통해 인식한 Tag 앨리먼트로는 attribute에 직접 접근 가능 (~~.width, ~.height ..)
   
  * property : JS 가 정의한 객체 내부에 정의된 속성 (style 의 속성 등)
   -> 1. 재산, 소유물
   -> 2. 부동산
   -> 3. 건물, 건물 구내 (부속건물 등)   

=> DOM 객체를 지칭   : node (속성: property) 
  * node 
     -> 1. (나무줄기의) 마디
   -> 2. (뿌리·가지의) 옹이
   -> 3. (연결망의) 교점
   
=> Tag 를 지칭 : element (속성: attribute) 
  * element
    -> 1. 요소, 성분
    -> 2. 약간의 놀라움, 위험, 진실 등
    -> 3. (집단·사회 내의 특정) 부류 
    
    => document.getElementById('test') 
   -> DOM 트리의 Node 를 인식 ( Tag 의 속성에 직접 접근 가능 )
